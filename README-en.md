# voicevox-mac-notificator

A tool to read out Mac notifications using [voicevox](https://github.com/VOICEVOX)

# how to use

```
$ git clone https://github.com/mkan0141/voicevox-mac-notificator
$ cd voicevox-mac-notificator
$ ./setup.sh
$ yarn start
```
